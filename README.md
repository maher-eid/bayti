# Bayti Lebnani React

## Run locally

```bash
npm install
npm run dev
```

## Included
- React + Vite setup
- React Router pages
- Shared store with localStorage persistence
- Home, category, product, cart, checkout, and admin pages
- Reused styling from the original project

## Main folders
- `src/components`
- `src/pages`
- `src/context`
- `src/data`
- `src/hooks`
- `src/styles`
